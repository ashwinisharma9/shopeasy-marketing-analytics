# ShopEasy Marketing Analytics
### Tools: SQL | Python | Power BI

## Business Problem
ShopEasy, an online retail business, faced declining customer engagement and conversion rates despite heavy investment in marketing campaigns. The goal was to identify the root causes and provide data-driven recommendations.

## Project Workflow
- **SQL** — Data cleaning, removing duplicates, handling missing values, and transforming raw marketing data
- **Python (Pandas & NLTK)** — Sentiment analysis on customer reviews to extract satisfaction trends
- **Power BI** — Built an interactive dashboard to visualize conversion rates, campaign performance, and customer engagement

## Key Insights
- Identified underperforming marketing campaigns impacting conversion rates
- Uncovered customer sentiment patterns from reviews
- Visualized the full customer journey to highlight drop-off points in the sales funnel

## Skills Demonstrated
SQL · Python · Pandas · NLTK · Power BI · Data Cleaning · Sentiment Analysis · Dashboard Design
